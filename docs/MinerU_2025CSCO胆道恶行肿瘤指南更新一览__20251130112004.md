# 引言

「2025CSCO指南会」于2025年4月18日至19日在济南山东大厦举办。本届大会由中国临床肿瘤学会（CSCO）、北京市希思科临床肿瘤学研究基金会联合主办，会议上发布了《2025CSCO胆道恶性肿瘤诊疗指南》更新（建议稿），并首次重磅发布了《胆道恶性肿瘤免疫检查点抑制剂专家共识》。其中，来自北京大学肿瘤医院周军教授、浙江省肿瘤医院应杰儿教授就《2025版CSCO：胆道恶性肿瘤诊疗指南更新（建议稿）》、《胆道恶性肿瘤免疫检查点抑制剂专家共识》进行了解读。

# PART 01

# 胆道恶性肿瘤诊疗指南更新和补充建议

# 1 胆管癌的筛查和诊断更新和补充建议

1）建议在【1.2胆管癌的筛查和诊断】章节的「b部分」之前，添加中国胆道癌(BTC）流行病学特点

中国BTC的疾病负担不可忽视。研究报道，超过  $60\%$  的中国BTC患者被诊断为晚期（III/IV期），约2/3不可切除。中国BTC的发生率和死亡率在老年人群中显著增加，中国城镇人口的BTC发生率高于农村区域。此外，肝外胆管癌（ECC）和肝内胆管癌（ICC）更常见于男性，而胆囊癌（GBC）更常见于女性。

2）建议在【1.2胆管癌的筛查和诊断】章节的「b部分」之后，添加胆管癌流行病学特点

胆管癌的危险因素包括胆管结石、肝硬化、化学暴露（如二噁英、亚硝胺）、原发性硬化性胆管炎、胆总管囊肿，以及乙型肝炎病毒（HBV）、丙型肝炎病毒（HCV）感染等。胆管癌风险因素之间复杂的相互作用尚不明确，这些因素在胆管癌发生发展中的重要性可能因人群和地理区域而异。

# 2 晚期胆道恶性肿瘤的一线治疗更新和补充建议

建议在【6.4晚期胆道恶性肿瘤的一线治疗】章节后的「注释」中，更新「24个月对照组OS率，添加KEYNOTE-966的3年OS率和中国亚组描述」。

KEYNOTE-966 研究中，治疗组 24 个月 OS 率估计值为  $25\%$  ，对照组为  $19\%$  ；治疗组 36 个月 OS 率估计值为  $13\%$  ，对照组为  $11\%$  。KEYNOTE-966 研究中国亚组结果显示帕博利珠单抗 + 吉西他滨联合顺铂组的中位 OS 为 14.1 个月，对照组 9.9 个月；中位 DoR 分别为 10.2 个月和 5.7 个月；ORR 分别为  $36\%$  和  $28.9\%$  。

TOPAZ-1 研究显示，度伐利尤单抗 + 吉西他滨联合顺铂将晚期 BTC 患者的 OS 由 11.3 个月提升至 12.9 个月（HR = 0.74，95% CI：0.63-0.87），36 个月 OS 率估计值由 6.9% 提升至 14.6%；PFS 从 5.7 个月提升至 7.2 个月，有效率分别为 26.7% vs. 18.7%

# 4 要点总结

1）胆道恶性肿瘤恶性程度高，晚期患者较多，需探索更多治疗方案，丰富晚期患者治疗选择；  
2）一线治疗以免疫联合化疗为核心，TOPAZ-1、KN966研究长期随访均支持其一线治疗应用的价值；

3）其他治疗手段中，HER2靶向治疗为精准治疗提供新方向，放化疗一线治疗或辅助治疗的价值有限，后线靶向治疗（如泽尼达妥单抗）显示治疗潜力，但仍需优化人群筛选及联合策略。

# 总结

2025年CSCO胆道恶性肿瘤诊疗指南重点强调了筛查、诊断、治疗及免疫检查点抑制剂共识。

指南预计新增中国胆道癌流行病学特点，强调晚期患者比例高，需重视筛查。

治疗方面，一线治疗建议推荐免疫联合化疗，KEYNOTE-966和TOPAZ-1研究支持其疗效。

二线治疗建议新增HER2阳性肿瘤推荐泽尼达妥单抗。

免疫治疗共识强调PD-1/PD-L1单抗联合化疗为一线首选，二线及后线治疗考虑ICI单药或联合策略。

除此之外，指南还探讨了新辅助免疫治疗、术后辅助治疗及免疫联合局部放疗等策略，并明确了MSI-H/dMMR为重要疗效预测标志物。

总体来说，此次更新有望进一步提升胆道恶性肿瘤诊疗水平，从而改善患者预后。
