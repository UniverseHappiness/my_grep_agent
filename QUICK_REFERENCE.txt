┌─────────────────────────────────────────────────────────────────────┐
│ ⌨️  常用快捷键                                                        │
└─────────────────────────────────────────────────────────────────────┘

  历史导航：
    ⬆️        上一条历史命令
    ⬇️        下一条历史命令
  
  光标移动：
    ⬅️        向左移动一个字符
    ➡️        向右移动一个字符
    Ctrl+A   跳到行首
    Ctrl+E   跳到行尾
  
  编辑操作：
    Backspace  删除前一个字符
    Delete     删除后一个字符
    Ctrl+W     删除前一个单词
    Ctrl+K     删除到行尾
    Ctrl+U     删除到行首
  
  智能功能：
    Tab       自动补全命令
    ➡️        接受建议（灰色提示文本）
  
  控制操作：
    Enter     执行命令
    Ctrl+C    取消当前操作
    Ctrl+D    退出程序

  history      查看搜索历史

┌─────────────────────────────────────────────────────────────────────┐
│ 💡 使用技巧                                                          │
└─────────────────────────────────────────────────────────────────────┘

  1. 快速重复命令
     按⬆️找到历史命令，直接Enter执行
  
  2. 修改历史命令
     按⬆️找到命令 → 用⬅️➡️移动 → 编辑 → Enter
  
  3. 自动补全
     输入he → 按Tab → 自动补全为help
  
  4. 智能建议
     开始输入时出现灰色提示 → 按➡️接受
  
  5. 快速编辑
     Ctrl+A跳开头 → Ctrl+K删到尾 → 重新输入

┌─────────────────────────────────────────────────────────────────────┐
│ 🐛 常见问题                                                          │
└─────────────────────────────────────────────────────────────────────┘

  Q: 方向键显示乱码？
  A: 重新安装：pip install --upgrade prompt_toolkit
  
  Q: 没有彩色显示？
  A: 使用支持ANSI颜色的现代终端
  
  Q: Tab补全没反应？
  A: 只能补全预定义命令（help、exit等）
  
  Q: 不想安装prompt_toolkit？
  A: 使用其他模式：python -m grep_agent -m enhanced
